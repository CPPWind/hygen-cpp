# hygen-contemplate
This is a hygen package for create-react-app (CRA) that supercharges your workflow.

## Installation
`$ cd project`
`$ yarn add --dev git+ssh://git@gitlab:4winds/hygen-contemplate`
`$ hygen-add contemplate`
 
## Ruby

### Graphql::Type
`$ hygen rg graphql-type --name`

### Graphql::QueryType
`$ hygen rg query --name`
